## Introduction

Welcome to easy-hooks-ts! React hook library, ready to use, written in Typescript.

## Overview

Easy Hooks TS is a collection of hooks and utilities for React. It is currently available for use in Node environments and works with any react meta framework like next, vite, astro. Support for SSR and browsers is coming soon.
